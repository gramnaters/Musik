## What is Musik?

**Musik** is a beautiful, privacy-focused music player built for those who appreciate minimalism and high-fidelity sound. Designed with a clean aesthetic and powerful features, it provides a seamless streaming experience across multiple themes (Apple, Spotify, and Tidal).

## Features

### Audio Quality & Experience

- **High Fidelity:** Support for HD, HiFi, and Dolby Atmos metadata tracking.
- **Immersive Visuals:** Dynamic album art backgrounds and custom seekbar styles.
- **Smooth Transitions:** Fluid animations for playback controls and UI shifts.
- **Audio Customization:** Integrated equalizer and audio settings.

### Interface & Themes

- **Multi-Theme Support:** Switch between Spotify, Apple Music, and Tidal aesthetics instantly.
- **Minimalist Design:** Focus on the music with a clutter-free, high-contrast interface.
- **Now Playing View:** A dedicated full-screen experience with focused controls.
- **Responsive Layout:** Optimized for both desktop and mobile devices.

### Library & Organization

- **Universal Search:** Find your favorite tracks across multiple sources and addons.
- **Personal Library:** Save and organize your favorite songs.
- **Queue Management:** Intuitive shuffle and repeat modes.
- **Explicit Tagging:** Clear indicators for explicit content.

## Installation

### Local Development

#### Prerequisites

- [Node.js](https://nodejs.org/) (Version 20+ recommended)
- [Bun](https://bun.sh/) (Optional, for faster performance)

#### Setup

1. **Clone the repository:**
    ```bash
    git clone https://github.com/gramnaters/Music-Player.git
    cd Music-Player
    ```

2. **Install dependencies:**
    ```bash
    npm install
    # or
    bun install
    ```

3. **Start the development server:**
    ```bash
    npm run dev
    # or
    bun run dev
    ```

4. **Open your browser:**
    Navigate to `http://localhost:3001/`

## Usage

### Keyboard Shortcuts

| Shortcut      | Action                       |
| ------------- | ---------------------------- |
| `Space`       | Play / Pause                 |
| `→`           | Seek forward 10s             |
| `←`           | Seek backward 10s            |
| `Shift` + `→` | Next track                   |
| `Shift` + `←` | Previous track               |
| `↑`           | Volume up                    |
| `↓`           | Volume down                  |
| `M`           | Mute / Unmute                |
| `S`           | Toggle shuffle               |
| `R`           | Toggle repeat                |
| `Q`           | Open queue                   |
| `L`           | Toggle lyrics                |
| `/`           | Focus search                 |
| `Esc`         | Close modals                 |
